

<form>: Define um formulário HTML. O atributo action especifica o URL de destino onde os dados do formulário serão enviados após o envio. 
O atributo method define o método HTTP usado para enviar os dados, que neste caso é "POST".



<label>: É usado para criar rótulos para os campos de entrada. 
O atributo for do label deve corresponder ao atributo id do campo de entrada associado para fornecer uma acessibilidade melhor para usuários com deficiência visual.



<input>: É o elemento de entrada. O atributo type define o tipo de campo de entrada, como "text" para campos de texto e "submit" para o botão de envio. 
O atributo id é usado para associar o rótulo (label) e o campo de entrada. O atributo name é o nome do campo que será usado quando o formulário for submetido para identificar o valor associado com esse campo.



<input type="submit" value="Registrar">: Este é o botão de envio do formulário. 
Quando o botão é clicado, o formulário será submetido e os dados serão enviados para o URL especificado no atributo action.
